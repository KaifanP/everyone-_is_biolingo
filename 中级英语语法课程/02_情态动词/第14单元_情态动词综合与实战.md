## 二、讲解（Presentation）

### 2.1 情态动词的语气强弱

不同情态动词的语气强弱不同。

#### 义务/必要性

```
must > have to > should > ought to > may > might
必须   不得不    应该    应该     可以   也许
```

#### 推测/可能性

```
must > may > might > could > can't
一定   可能   也许   可能   不可能
```

#### 能力

```
can > could > be able to
能   能     能够
```

#### 许可

```
may > can > could
可以  可以   可以(礼貌)
```

### 2.2 情态动词的多重含义

很多情态动词有多种含义，需要根据语境判断。

#### can的三种用法

1. **能力**：I **can** swim.
2. **许可**：You **can** go now.
3. **可能性**：It **can** be very cold in winter.

#### must的两种用法

1. **义务**：You **must** go now.
2. **推测**：He **must** be tired.

#### may的两种用法

1. **许可**：You **may** leave.
2. **可能性**：He **may** be at home.

#### will的多种用法

1. **意愿**：I **will** help you.
2. **将来**：I **will** go tomorrow.
3. **习惯**：He **will** sit for hours.
4. **请求**：**Will** you help me?

### 2.3 情态动词的时态变化

| 现在 | 过去 | 含义 |
|------|------|------|
| can | could | 能力/许可 |
| may | might | 许可/可能 |
| must | had to | 必须 |
| will | would | 意愿/习惯 |
| shall | should | 应该 |

**例句对比**：
- I **can** swim now.（我现在会游泳。）
- I **could** swim when I was young.（我年轻时会游泳。）

- You **may** leave now.（你现在可以走了。）
- You **might** leave if you wanted.（如果你想走，你可以走。）

- I **must** go now.（我现在必须走。）
- I **had to** go yesterday.（我昨天不得不走。）

### 2.4 情态动词+完成时的用法

| 结构 | 含义 | 例句 |
|------|------|------|
| must have done | 一定做了 | He must have left. |
| may have done | 可能做了 | He may have left. |
| might have done | 也许做了 | He might have left. |
| could have done | 可能做了/本可以 | He could have left. |
| can't have done | 不可能做了 | He can't have left. |
| should have done | 应该做但没做 | You should have told me. |
| shouldn't have done | 不应该做但做了 | You shouldn't have told him. |
| needn't have done | 本不必做但做了 | You needn't have waited. |
| would have done | 本会做（虚拟） | I would have gone if invited. |

## 三、辨析（Analysis）

### 3.1 高频易错点

#### 易错点1：must的否定

- 表示推测时：He **can't** be at home.（不可能在家）
- 表示义务时：You **mustn't** go.（不能走）

#### 易错点2：could的多种含义

- 能力：I **could** swim when I was young.
- 许可：**Could** I use your phone?
- 可能：He **could** be at home.
- 虚拟：I **could have helped** if I had known.

#### 易错点3：should have done的时态

- ✗ You **should told** me.
- ✓ You **should have told** me.

#### 易错点4：混淆needn't have done和didn't need to

- needn't have done：做了但不必做
- didn't need to：不必做（通常没做）

### 3.2 语境判断技巧

如何判断情态动词的含义？

**技巧1：看动词类型**
- 状态动词（be, have, know等）→ 通常是推测
- 动作动词（go, do, make等）→ 通常是义务/意愿

**技巧2：看上下文**
- 有证据 → 推测
- 有规则/要求 → 义务
- 有请求 → 许可/能力

**技巧3：看时间**
- 现在 → 情态动词 + 动词原形
- 过去 → 情态动词 + have + 过去分词

## 六、总结（Summary）

### 核心要点

1. **情态动词的语气强弱**：
   - 义务：must > have to > should > may
   - 推测：must > may > might > could > can't

2. **情态动词的多重含义**：
   - can：能力/许可/可能
   - must：义务/推测
   - may：许可/可能
   - will：意愿/将来/习惯

3. **情态动词+完成时**：
   - must have done：一定做了
   - should have done：应该做但没做
   - could have done：本可以/可能做了

4. **过去推测**：
   - must have done：一定做了
   - can't have done：不可能做了

### 情态动词总表

| 情态动词 | 用法 | 例句 |
|----------|------|------|
| can | 能力/许可/可能 | I can swim. |
| could | 过去能力/礼貌请求/可能 | I could swim when I was young. |
| may | 许可/可能 | You may leave. |
| might | 可能（不确定） | He might be at home. |
| must | 义务/推测 | You must go. / He must be tired. |
| have to | 客观义务 | I have to go. |
| should | 建议/应该 | You should go. |
| ought to | 正式建议 | You ought to go. |
| will | 意愿/将来/习惯 | I will help you. |
| would | 过去习惯/虚拟/礼貌 | He would sit for hours. |
| need | 需要 | You needn't go. |
| dare | 敢 | How dare you! |
